# SwipeCard コンポーネント デザイン仕様書

## 概要
- 用途: 作品選択UI（Tinder風カード）
- 対象画面: 作品発見（Discover）ページ
- 設計担当: 玉田（第1中隊デザイン）

## インタラクション

| ジェスチャー | アクション | フィードバック |
|------------|----------|--------------|
| 右スワイプ | お気に入り登録 | 「LIKE」ラベル表示（緑） |
| 左スワイプ | 興味なし（スキップ） | 「NOPE」ラベル表示（赤） |
| ダブルタップ | 即座に読み始める | 画面遷移 |

## サイズ

| プロパティ | 値 | 備考 |
|---------|-----|------|
| 幅 | 327px | コンテナ幅 - padding 24px * 2 |
| 高さ | 476px | カバー220px + コンテンツ256px |
| 角丸 | 16px (--radius-swipe) | カード角丸 |
| シャドウ | --shadow-card | 通常時 |


## スワイプ動作仕様

| パラメータ | デフォルト値 | 説明 |
|----------|------------|------|
| swipeThreshold | 100px | スワイプ発動の最小距離 |
| ROTATION_FACTOR | 0.1 | ドラッグ距離に対する回転角度の係数 |
| OPACITY_FACTOR | 0.003 | LIKE/NOPE表示の不透明度係数 |
| 退出アニメーション | 300ms | カードが画面外に出る時間 |

## 状態

| 状態 | 見た目 | トランジション |
|------|-------|--------------|
| Default | 静止、cursor: grab | - |
| Dragging | 位置追従、回転、cursor: grabbing | なし（即座追従） |
| Like表示中 | 右寄り + 緑「LIKE」 | opacity fade |
| Nope表示中 | 左寄り + 赤「NOPE」 | opacity fade |
| Exiting | 画面外へ飛ぶ | 300ms ease-out |
| Snap back | 元位置に戻る | 300ms ease-gentle |

## アクセシビリティ

- role="button", tabIndex={0}
- aria-label="スワイプして評価"
- キーボード操作: 今後実装予定（左右矢印キー）

## Props

| Prop | 型 | デフォルト | 説明 |
|------|-----|----------|------|
| onSwipe | (direction: "left" \| "right") => void | - | スワイプ完了時のコールバック |
| onDoubleTap | () => void | - | ダブルタップ時のコールバック |
| swipeThreshold | number | 100 | スワイプ発動の閾値(px) |
| swipeable | boolean | true | スワイプ操作の有効/無効 |
| className | string | - | 追加CSSクラス |
| children | ReactNode | - | カードの中身 |
